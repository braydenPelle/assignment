# Assignment

Create an endpoint that parses a list of JSON files and replaces a limited number of "dog" values with "cat".

## Table of Contents

*   [Setup](#setup)


## Setup


1.  Clone the repository
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Compile TypeScript:
    ```bash
    npm run build
    ```
4. Run tests:
    ```bash
    npm run test
    ```
5. Start API:
    ```bash
    npm run start
    ```
